# Tasty Bites Restaurant API

## Description
Express.js REST API for managing restaurant menu items.

Includes:
- Full CRUD endpoints
- Custom logging middleware
- Validation middleware using express-validator
- Proper HTTP status codes

## Endpoints
GET /api/menu
GET /api/menu/:id
POST /api/menu
PUT /api/menu/:id
DELETE /api/menu/:id
